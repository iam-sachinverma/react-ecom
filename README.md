## 💻 Getting started

### Requirements

- Have NodeJS
- Have a Firebase account

**Clone the project and access the folder**

```bash
$ git clone https://github.com/alxrdev/crwn-clothing.git
```

**Follow the steps below**
```bash
# Install the dependencies
$ npm install

# Start the client
$ npm start
```
